# Client Experience V2.1 Founder Checklist

Branch: `design/client-experience-v2-1`

Production and `main` are unchanged.

## Review path
1. Complete the Hub priority actions.
2. Search for a customer and add a note.
3. Filter the calendar and confirm an appointment.
4. Open a conversation, edit the suggested reply and send it.
5. Change an invoice state.
6. Reconcile a bank-feed item.
7. Filter and inspect a document.
8. Change report controls.
9. Switch between Owner and Staff.
10. Apply every Theme Lab preset.
11. Create a custom theme and test the contrast warning.
12. Enable reduced motion.
13. Go offline, make local changes, then reconnect.
14. Test mobile navigation and quick-create forms.
15. Reset the preview from Founder Review Mode.

## Production work still required
- Persist approved workspace themes.
- Enforce roles and feature access on the server.
- Implement real local storage, retries and sync conflicts.
- Connect production data, files and external services.
- Complete accessibility, performance and real-user testing.

No migration occurs until Giovanni, Matthew and Niki approve the direction and migration plan.
